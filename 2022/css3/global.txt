

  *properties shorthand: s{all:;}
  
  *selector{property|all: initial;}
    •set initial value to property based on W3C spec'ification.
    •ex: table{display: initial;} → display:inline'd; ≠ not display:table; !!!
  
  *s{p|all: inherite;}
    •set closest parent value
  
  *s{p|all: unset;}
    •set 'inherited value' else if(non-inherited){set 'initial value'}
  
  *s{p|all: revert;}
    •ignore current declaration-block properties, set 'previos auther-origin styles' else{'user-agent-origin styles'}
      »UA-origin: based on collection of rules (not based on spec).
    •ex:
      div.a{display: grid;} /*previos author-origin style*/
      /*.............*/
      div.a, div.b{display:flex; display:revert;} →↓
       → .a{display:grid;} /*set previos A-origin style*/
       → .b{display:block;} /*set UA-origin style(not initial)*/
