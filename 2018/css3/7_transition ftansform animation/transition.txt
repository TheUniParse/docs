

★transitions: 
  •applay to all elements && ::before && ::after.
    »do not applay it to states(el:hover …)
  •required :hover or :focuse or :active (state pseudo class) or java sceipt (onClick, onDblClick, onScroll, onInput, onChange …)
  *transition-property:all'd|property-name|none;
    »targeting any property have value of length|number|color.
    
  *transition-duration|transition-delay:#s'0d|#ms;
  
  *transition-timing-function:
     ease'd|ease-in-out|ease-in|ease-out
     |linear|cubic-bezier(x2,y2,x3,y3)
     |step-start|step-end|steps(#, start|end)
    »ease: slow at start & end, (at middle have change from slow to fast)
    »ease-in-out: like ease,but at middle have non-change (linear)
    »ease-in: slow at start.
    »ease-out: slow at end.
    »linear: non-change.
    »cubic-bezier(x2,y2,x3,y3): creat custom curve,
       x€[0,1] y€[0,+∞[
      •Point2(x2,y2) creat touching-line with Point1(0,0),
      •Point3(x3,y3) creat touching-line with Point4(1,1).
    »steps(#, start|end):
       #: number of steps, devide duration to fractions, 
       start:set initial value of properties on every 1step done
       end: set final value of properties on every 1step done.
      •step-start: steps(1, start)
      •step-end: steps(1, end)

  *shorthand: 
    transition: property(r) 'duration(r) delay' timing-function.
  
  *multitransitions:
    »individual:
      t-property: p1, p2, …;
      t-duration: dr1, dr2, …;
      t-delay: dl1, dl2, dl3 …;
      t-timing-function: tf1, tf2, …;
    »shorthand:
      transition: p1 dr1 dl1 tf1, p2 dr2 dl2 tf2, …;
    •Note1: at individual, omitting values do-not return them default !!!
      instead, refill omitted values with the already list-values by order from the first one to second, and so on.
      ›ex: 
       t-property: p1, p2, p3, p4, p5, p6, p7;
       t-duration: d1, d2, d3; → t-d: d1,d2,d3,d1,d2,d3,d1;
    •Note2: at shorthand, omitted values returns to default.
      ›ex:
       transition: all .2s 150ms cubic-bezier(.2, 5, .7, .5),
                   left .9s steps(10, end);↓
       →transition: all .9s 150ms cubic-bezier(.2, 5, .7, .5),
                   left .2s 0 steps(10, end);
